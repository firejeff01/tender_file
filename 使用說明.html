標案文件產生器

行政人員使用說明

這個工具會依照 Excel 裡填的標案資料，自動產生 保密切結書、保密同意書、著作權同意書等文件（ODT 格式）， 並把公司名稱、負責人、電話、地址、機關名稱、案名、簽署日期等欄位自動填好。
免安裝——把資料夾複製到哪都能用，雙擊 exe 即可。

這個資料夾裡有什麼

  • 標案文件產生器.exe要執行的程式（雙擊它）
  • 標案資料.xlsx你要填寫的 Excel
  • 範本\文件範本（請勿更動裡面的檔案）
  • 使用說明.txt／.html就是本說明（.html 版用瀏覽器開較好讀）
💡 輸出\ 與 設定.txt 會在你開始使用後由程式自動產生，不用理會。

第一次使用只需做一次

  1. 打開 「標案資料.xlsx」。
  2. 切到 「公司資料」 工作表，填好：廠商名稱、負責人姓名、聯絡電話、廠商地址。
    • 這些是公司固定資料，之後每個標案都會自動帶入，不用再填。
  3. 存檔。

每接一個新標案

  1. 打開「標案資料.xlsx」，切到 「標案清單」 工作表。
  2. 在最下面新增一列，填寫：
    • 標案名稱：必填。這會變成輸出資料夾的名字。
    • 機關名稱：招標的機關，例如「花蓮縣政府」。
    • 案件名稱：文件裡引用的正式案名（留空 → 直接用「標案名稱」）。
    • 簽署日期：民國格式，如 115年8月1日 或 115/8/1（留空 → 用產生當天）。
    • 產生:○○○ 這幾欄：選「是」或「否」，決定要產生哪些文件（留空視同「是」）。
    • 備註：自由填寫，不會出現在文件裡。
  3. 存檔，並關閉 Excel（重要，不然程式可能讀不到最新內容）。
  4. 雙擊 「標案文件產生器.exe」。
  5. 在清單中勾選要產生的標案（預設會自動勾選還沒產生過的），按 ✔ 產生文件。
  6. 完成後可按 📂 開啟輸出資料夾 查看。文件在 （輸出資料夾）\（你的標案名稱）\。
📁 輸出資料夾預設是程式旁的「輸出」。想改到別的位置（例如共用磁碟），按畫面上方「輸出資料夾」右邊的 變更…，程式會記住，下次開啟仍沿用。

範本管理：新增／移除文件範本進階

以後若要增減文件種類，不用找工程師。開程式後按上方 🗂 範本管理…， 即可看到目前所有範本與它們用到的參數，並可新增或移除。

新增一種文件範本

  1. 準備一份該文件的 ODT，把「要每次替換的文字」標上顏色：
    • 紅色＝公司固定資料（公司名、負責人、電話、地址）
    • 藍色＝每案不同的資料（機關、案名、日期…）
    （顏色只是給工具辨識用，產生出來的正式文件會是黑字。）
  2. 在「範本管理」按 ＋ 新增範本…，選那份 ODT。工具會列出所有彩色片段並盡量幫你對好參數：
    • 已知的公司資料會自動對應（顯示廠商名稱、聯絡電話…）。
    • 新的欄位會顯示「（請選擇）」，可從下拉選現有參數，或選「＋ 新增參數…」自己命名（並指定「每案不同」或「公司固定」）。
  3. 填範本名稱，按 ✔ 建立範本。工具會把範本存進「範本」資料夾，並自動在 Excel 加上需要的欄位。之後這份文件就會跟其他文件一樣出現在清單裡。

移除一種文件範本

在「範本管理」選一份範本，按 🗑 移除選取範本。工具會先列出將刪除的 Excel 欄位讓你確認，確定後就刪掉範本並同步移除它專屬的欄位。
（只被這份範本用到的欄位才會刪；公司名稱、機關等共用欄位會保留。）

⚠️ 新增或移除需要更新 Excel。若 Excel 正開著「標案資料.xlsx」，工具會跳提示「將關閉 Excel」， 請先在 Excel 存檔，按確定後工具會自動關閉 Excel 並繼續（若 Excel 詢問是否儲存，依需要選擇即可）。

常見問題

按了沒反應／顯示紅色錯誤？
紅色那一列的「訊息」欄會寫原因，最常見是「公司資料」還沒填、或簽署日期格式打錯。修正 Excel、存檔、按 🔄 重新讀取 Excel 即可。
同一個標案想重做？
勾選那一列，勾下方 「重新產生（先清空該標案資料夾再重產）」，再按產生。這會先把該標案的輸出資料夾清空，再產生目前選取的文件——所以之前產生、後來取消勾選的舊檔不會殘留。
不勾的話，已經產生過的檔案會被略過、不會覆蓋（只補產生缺少的）。
某個標案要用不同公司投標？
在「標案清單」右側的 廠商名稱／負責人姓名／聯絡電話／廠商地址 欄位，個別填入該列要用的值；有填就會優先用你填的，沒填才用「公司資料」的預設值。
可以用不同公司電腦跑嗎？
可以，整個資料夾複製過去即可，不需要安裝任何東西。（需要 Windows 10（1903 以上版本）或 Windows 11。）
兩個人同時對同一個標案產生？
請避免。若同時產生同一個標案，後存的會蓋掉先存的。

重要提醒

⛔ 「範本」資料夾裡的檔案請「不要」用 Word 或 LibreOffice 打開後另存， 那會破壞裡面的參數標記，導致之後產生的文件出錯。
需要新增／修改文件種類時，請用程式內的「範本管理」功能，或聯絡資訊人員。
